# 📘 Projetos de Extensão de Backend

🏫 Projeto desenvolvido na Unicarioca como avaliação instituicional

📆 Data: 23/06/26

📋 Objetivo: Fazer um sistema web para auxiliar no gerenciamento de projetos voltados às escolas públicas, permitindo o cadastro, visualização e organização de eventos educacionais.


## 🚀 Funcionalidades

- Cadastro de projetos educacionais;
- Visualização da lista de projetos cadastrados;
- Consulta de detalhes dos projetos;
- Deletar projetos;
- Atualizar projetos;
- Interface intuitiva e responsiva.
